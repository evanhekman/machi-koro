## purpose
Python prototyping layer. experiment with new ideas (pruning heuristics, income models,
strategy comparisons) here before committing to a Rust implementation.

## files
- `analysis.py` — full Python reimplementation of the expectimax solver; mirrors Rust logic
- `simulate.py` — runs games with a given strategy function
- `strategies.py` — heuristic baselines to benchmark against the solver
